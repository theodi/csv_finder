# CSV Finder

A quick script that hoovers up all the CSVs on [data.gov.uk](http://data.gov.uk). Includes the data as of 17th January 2014.

To run locally:

* Clone the repo
* Run `bundle install`
* Run `ruby lib/csv_finder.rb`
